# Hướng Dẫn Cài Đặt

## 1. Tạo môi trường ảo

```bash
python -m venv .venv
```

## 2. Kích hoạt `.venv`

```bash
.venv\Scripts\Activate.ps1
```

## 3. Cài thư viện

```bash
pip install -r requirements.txt
```

## 4. Chạy ứng dụng

```bash
python run_app.py
```

## 5. Test nhanh CLI

```bash
python run_detect.py --mode auto
```
